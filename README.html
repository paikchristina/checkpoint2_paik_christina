Project Specs:

Prepros:

To utilize this repository, Prepros is needed for the SCSS files to compile and also minify and optimize the CSS and JavaScript. Prepros is free, easy to use, and can be downloaded here.

SCSS:

In this repository, SCSS is utilized. Please keep this in mind when viewing or editing the code!

jQuery Libraries:

  • jQuery - Up-to-date jQuery file from the source.
  • Owl Carousel - Image galleries and automatic scrolls; used for image gallery on "About" page.
  • Fit Vids - Easy responsive embedded video; used on "Blog Post" page.

Other Resources:

  • Hover.css - Clean and simple hover animations with no JavaScript involved; used for the logo.
